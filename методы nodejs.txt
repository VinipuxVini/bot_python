Команды и хендлеры
start — /start — Приветствие и вывод справки по командам.
admin_login — /admin_login — Ввод 4-значного кода администратора для получения прав.
set_description — /set_description — Установка описания для файлов.
send_file — /send_file — Запрос на отправку файла в группу.
send_media — /send_media — Запрос на отправку медиа-файла (фото/видео) с подписью.
announcement — /announcement — Отправка текстового объявления во все группы.
get_report — /get_report — Получение отчета по отправленным файлам за выбранную неделю.
group_list — /group_list — Получение списка всех групп, в которых состоит бот.
delete_last — /delete_last — Удаление последнего отправленного сообщения или файла в одной группе.
delete_last_all — /delete_last_all — Удаление последних сообщений или файлов во всех группах.
Inline-хендлеры (кнопки)
select_week_N — (кнопка) — Выбор недели для просмотра отчета.
prev_page_N / next_page_N — (кнопки) — Переключение страниц с неделями отчетов.
select_{id}_{index} — (кнопка) — Выбор группы для отправки файла, если найдено несколько совпадений.
cancel_{id} — (кнопка) — Отмена отправки файла при выборе группы.
confirm_delete_all — (кнопка) — Подтверждение удаления всех последних сообщений во всех группах.
cancel_delete_all — (кнопка) — Отмена удаления всех последних сообщений.
Событийные хендлеры
on("document") — (файл) — Обработка загруженного документа, автоматический или ручной выбор группы для отправки.
on("text") — (текст) — Обработка текстовых сообщений: описание, объявление, подпись к медиа.
on(["photo", "video"]) — (медиа) — Обработка фото/видео для массовой рассылки.
on("my_chat_member") — (событие) — Добавление/удаление бота из группы, уведомление администраторов.
on("new_chat_title") — (событие) — Изменение названия группы, уведомление администраторов.
Вспомогательные функции
sendFileToGroup — (нет команды) — Отправка файла в выбранную группу.
sendMediaGroup — (нет команды) — Отправка медиа-группы во все группы.
sendBatchReport — (нет команды) — Отправка итогового отчета по текущей загрузке файлов.
generateReportMessage — (нет команды) — Формирование текста отчета по неделе.
sendWeeklyReport — (нет команды) — Автоматическая отправка отчета администраторам по расписанию.
checkAdmin — (нет команды) — Проверка, является ли пользователь администратором.
